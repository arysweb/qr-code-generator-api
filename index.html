GET /generate
GET /generate?data=https://example.com&size=300&format=png&dark=%23000000&light=%23ffffff Returns a PNG image directly (binary). For Base64 or SVG use format=base64 or format=svg.
POST /generate
POST /generate Content-Type: application/json { "data": "https://example.com", "size": 300, "format": "base64", "dark": "#000000", "light": "#ffffff", "margin": 2, "error_correction": "M" } Response: { "format": "base64", "data": "data:image/png;base64,iVBORw0KGgo...", "size": 300, "dark": "#000000", "light": "#ffffff", "margin": 2, "error_correction": "M", "content": "https://example.com" }
